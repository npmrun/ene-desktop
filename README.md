## 调试
1. 执行`npm run dev`
2. 打开`edge`浏览器
3. 输入`edge://inspect`并打开
4. 点击`Open dedicated DevTools for Node`
5. 在弹出的窗口处点击添加连接`localhost:5858`
6. 等待程序运行之后，左上角切换到源代码，就可以调试了
